Headless Mode API

Headless Mode allows binding QUploader triggers directly to custom DOM elements without rendering default uploader containers, file lists, or status bars.

1. File Selection Trigger

Trigger file pickers on button click:

$('#btn').quploader('browseFile', {
  autoUpload: false,
  onPick: (files) => { ... }
});
2. Folder Selection Trigger

Trigger folder structure pickers:

$('#btn').quploader('browseFolder', {
  autoUpload: false,
  allowFolder: true,
  onPick: (files) => { ... }
});
3. WebRTC Camera Trigger

Trigger WebRTC camera capture modal:

$('#btn').quploader('camera', {
  autoUpload: false,
  onPick: (files) => { ... }
});
4. Drag & Drop Area Trigger

Make any container a drag-and-drop zone:

$('#dropzone').quploader('dropzone', {
  autoUpload: false,
  onPick: (files) => { ... }
});
